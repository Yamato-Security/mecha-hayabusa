## 🦅 About

Mecha Hayabusa is an **AI analyzer, DFIR timeline and report generation tool** for
[Hayabusa](https://github.com/Yamato-Security/hayabusa) results. It exposes Hayabusa results to AI
assistants (such as Claude) over the **Model Context Protocol (MCP)**, so you can load datasets,
query and search, build timelines and analytics, drill into event details, decode artifacts and
extract IOCs — all in natural language.
