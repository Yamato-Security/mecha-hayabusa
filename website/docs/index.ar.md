## لماذا Mecha Hayabusa؟

<div class="grid cards" markdown>

-   :material-robot:{ .lg .middle } __تحقيق مدعوم بالذكاء الاصطناعي__

    ---

    استكشف نتائج [Hayabusa](https://github.com/Yamato-Security/hayabusa) باستخدام مساعد ذكاء اصطناعي
    باللغة الطبيعية، عبر **Model Context Protocol** (مثل Claude).

-   :material-database-cog:{ .lg .middle } __عمليات مجموعات البيانات__

    ---

    قم بتحميل مجموعات بيانات نتائج Hayabusa وإدارتها والعمل عليها حتى يتمكن الذكاء الاصطناعي من الاستدلال عليها.

-   :material-magnify:{ .lg .middle } __الاستعلام والبحث__

    ---

    استعلم وابحث عبر النتائج للتركيز على الأحداث المهمة.

-   :material-timeline-clock:{ .lg .middle } __الجدول الزمني والتحليلات__

    ---

    أنشئ جداول زمنية وقم بإجراء تحليلات لفهم شكل الحادث.

-   :material-shield-bug:{ .lg .middle } __تحليل التفاصيل ومؤشرات الاختراق (IOC)__

    ---

    تعمّق في تفاصيل الأحداث، وفك تشفير القطع الأثرية (مثل PowerShell المشفّر بـ Base64)، واستخرج مؤشرات الاختراق (IOC).

-   :material-file-document-edit:{ .lg .middle } __إنشاء التقارير__

    ---

    أنشئ جداول زمنية وتقارير DFIR من نتائج Hayabusa الخاصة بك.

</div>
