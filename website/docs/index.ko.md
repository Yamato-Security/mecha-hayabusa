## 왜 Mecha Hayabusa인가?

<div class="grid cards" markdown>

-   :material-robot:{ .lg .middle } __AI 기반 조사__

    ---

    **Model Context Protocol**을 통해 AI 어시스턴트(예: Claude)로 [Hayabusa](https://github.com/Yamato-Security/hayabusa) 결과를
    자연어로 탐색하세요.

-   :material-database-cog:{ .lg .middle } __데이터셋 작업__

    ---

    AI가 추론할 수 있도록 Hayabusa 결과 데이터셋을 로드, 관리 및 작업하세요.

-   :material-magnify:{ .lg .middle } __쿼리 및 검색__

    ---

    결과 전체를 쿼리하고 검색하여 중요한 이벤트에 집중하세요.

-   :material-timeline-clock:{ .lg .middle } __타임라인 및 분석__

    ---

    타임라인을 구축하고 분석을 실행하여 사건의 양상을 파악하세요.

-   :material-shield-bug:{ .lg .middle } __상세 및 IOC 분석__

    ---

    이벤트 세부 정보를 파헤치고, 아티팩트(예: Base64 PowerShell)를 디코딩하며, IOC를 추출하세요.

-   :material-file-document-edit:{ .lg .middle } __리포트 생성__

    ---

    Hayabusa 결과로부터 DFIR 타임라인과 리포트를 생성하세요.

</div>
