## ဘာကြောင့် Mecha Hayabusa နည်း။

<div class="grid cards" markdown>

-   :material-robot:{ .lg .middle } __AI စွမ်းအားသုံး စုံစမ်းစစ်ဆေးခြင်း__

    ---

    [Hayabusa](https://github.com/Yamato-Security/hayabusa) ရလဒ်များကို **Model Context Protocol** မှတစ်ဆင့် AI လက်ထောက်တစ်ခု (ဥပမာ Claude) ဖြင့် သဘာဝဘာသာစကားဖြင့် ရှာဖွေလေ့လာပါ။

-   :material-database-cog:{ .lg .middle } __Dataset လုပ်ဆောင်ချက်များ__

    ---

    Hayabusa ရလဒ် dataset များကို တင်သွင်း၊ စီမံခန့်ခွဲနှင့် လုပ်ဆောင်ခြင်းဖြင့် AI က ၎င်းတို့အပေါ် ဆင်ခြင်သုံးသပ်နိုင်စေသည်။

-   :material-magnify:{ .lg .middle } __Query နှင့် ရှာဖွေခြင်း__

    ---

    အရေးပါသော အဖြစ်အပျက်များကို တိကျစွာ ရှာဖွေနိုင်ရန် ရလဒ်များတစ်လျှောက် query လုပ်ပြီး ရှာဖွေပါ။

-   :material-timeline-clock:{ .lg .middle } __အချိန်ဇယားနှင့် ပိုင်းခြားစိတ်ဖြာချက်__

    ---

    ဖြစ်စဉ်တစ်ခု၏ ပုံသဏ္ဍာန်ကို နားလည်နိုင်ရန် အချိန်ဇယားများ တည်ဆောက်ပြီး ပိုင်းခြားစိတ်ဖြာချက်များ လုပ်ဆောင်ပါ။

-   :material-shield-bug:{ .lg .middle } __အသေးစိတ်နှင့် IOC ပိုင်းခြားစိတ်ဖြာချက်__

    ---

    အဖြစ်အပျက် အသေးစိတ်များကို တိုးဝင်လေ့လာ၊ artifact များကို ကုဒ်ဖြည်ခြင်း (ဥပမာ Base64 PowerShell)၊ နှင့် IOC များကို ထုတ်ယူပါ။

-   :material-file-document-edit:{ .lg .middle } __အစီရင်ခံစာ ထုတ်လုပ်ခြင်း__

    ---

    သင်၏ Hayabusa ရလဒ်များမှ DFIR အချိန်ဇယားများနှင့် အစီရင်ခံစာများ ထုတ်လုပ်ပါ။

</div>
