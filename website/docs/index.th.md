## ทำไมต้อง Mecha Hayabusa?

<div class="grid cards" markdown>

-   :material-robot:{ .lg .middle } __การสืบสวนที่ขับเคลื่อนด้วย AI__

    ---

    สำรวจผลลัพธ์ของ [Hayabusa](https://github.com/Yamato-Security/hayabusa) ด้วยผู้ช่วย AI ใน
    ภาษาธรรมชาติ ผ่าน **Model Context Protocol** (เช่น Claude)

-   :material-database-cog:{ .lg .middle } __การดำเนินการกับชุดข้อมูล__

    ---

    โหลด จัดการ และดำเนินการกับชุดข้อมูลผลลัพธ์ของ Hayabusa เพื่อให้ AI สามารถใช้เหตุผลกับมันได้

-   :material-magnify:{ .lg .middle } __การคิวรีและการค้นหา__

    ---

    คิวรีและค้นหาในผลลัพธ์เพื่อเจาะจงไปยังเหตุการณ์ที่สำคัญ

-   :material-timeline-clock:{ .lg .middle } __ไทม์ไลน์และการวิเคราะห์__

    ---

    สร้างไทม์ไลน์และทำการวิเคราะห์เพื่อทำความเข้าใจรูปแบบของเหตุการณ์

-   :material-shield-bug:{ .lg .middle } __การวิเคราะห์รายละเอียดและ IOC__

    ---

    เจาะลึกรายละเอียดของเหตุการณ์ ถอดรหัสอาร์ติแฟกต์ (เช่น Base64 PowerShell) และดึงข้อมูล IOC

-   :material-file-document-edit:{ .lg .middle } __การสร้างรายงาน__

    ---

    สร้างไทม์ไลน์และรายงาน DFIR จากผลลัพธ์ Hayabusa ของคุณ

</div>
