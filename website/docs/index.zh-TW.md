## 為什麼選擇 Mecha Hayabusa？

<div class="grid cards" markdown>

-   :material-robot:{ .lg .middle } __AI 驅動的調查__

    ---

    透過 **Model Context Protocol**（例如 Claude），以 AI 助理用自然語言探索 [Hayabusa](https://github.com/Yamato-Security/hayabusa) 結果。

-   :material-database-cog:{ .lg .middle } __資料集操作__

    ---

    載入、管理並操作 Hayabusa 結果資料集，讓 AI 能對其進行推理。

-   :material-magnify:{ .lg .middle } __查詢與搜尋__

    ---

    跨結果進行查詢與搜尋，鎖定真正重要的事件。

-   :material-timeline-clock:{ .lg .middle } __時間軸與分析__

    ---

    建立時間軸並執行分析，以掌握事件的全貌。

-   :material-shield-bug:{ .lg .middle } __細節與 IOC 分析__

    ---

    深入鑽研事件細節、解碼產物（例如 Base64 PowerShell），並擷取 IOC。

-   :material-file-document-edit:{ .lg .middle } __報告產生__

    ---

    從你的 Hayabusa 結果產生 DFIR 時間軸與報告。

</div>
